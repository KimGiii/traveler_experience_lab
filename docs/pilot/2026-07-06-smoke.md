### diagnose → scenario 핸드오프 검증

diagnose가 도출한 상위 마찰 중 2건이 **오늘 scenario 스모크의 실측 데이터로 즉시 근거화됨**:
- "확정/재고 불확실" ↔ TNA `date_unavailable` 실측 (voc-024류 마찰의 상품측 증거)
- "취소 정책 불투명" ↔ stay `free_cancellation` 필드로 상품별 판별 가능 (voc-020/021의 개선 시나리오 입력)

→ 두 wedge의 연결(diagnose 출력 → scenario 입력)이 실데이터로 동작함을 확인.
