## [T1 | 00:39:57] 문제 정의

## [T3 | 00:42:36] 엣지케이스 + 실험/QA 초안

**엣지케이스 6종 (전건 데이터 근거):**
- **취소/환불**: 숙소는 `free_cancellation=True` 실측. 반면 TNA 조인투어는 예약 전 응답에 취소정책 문구 없음(`cancellation_note=null`) — 카테고리 간 취소 정보 비대칭이 결제 전 화면에 드러나야 함.
- **재고없음**: 숙소 `available=True`·투어 11-07 가용 실측은 **조회 시점 값** — 조인투어는 정원 마감 리스크. 확정 직전 재조회 필요.
- **날짜불일치**: `getTnaOptions`는 `selectedDate` 단위 응답("2026-11-07 예약 가능합니다!") — 11-08로 옮기면 별도 확인 필요. 항공가는 11-06 출발 기준.
- **가격변동**: 숙소 원가 194,431→157,400원 할인 관측(`price_change` 플래그) — 할인 종료 시 1인 총액 +37,031원/박×3. 항공도 fixture `price_change`.
- **리뷰신뢰성**: 숙소 4.6/1,272건은 표본 충분. TNA 조인투어는 ⭐4.8이나 **리뷰수 미노출**(`review_count=null`) — 평점 단독으론 신뢰 판단 불가.
- **모바일**: 전 예약 URL 모바일 웹(`mobile` 플래그) — 부모님에게 링크 공유 시 3인 인원·확정 상태가 화면에 유지되는지가 UX 핵심.

**실험 초안 — "확정 상태 라이브 배지"**: 상품 카드·결제 전 화면에 `해당 날짜 예약 가능(조회 시각)`·`무료취소`·`취소정책 미제공` 배지를 실데이터로 노출. 가설: 확정 민감 여행자(부모님 동반)의 결제 진입률 상승·"예약 확정되나요" 문의 감소. 지표: 결제 진입률, 예약확정 CS 문의율, 결제 전 이탈률. 가드레일: 배지-실제 확정 상태 불일치율. 실패조건: 불일치 >2% 또는 문의율 개선 없이 이탈만 감소.

**QA 확인 항목**: 전 URL에 3인(adultCount=3) 유지, 무료취소 배지-실제 규정 일치, 11-06~09 날짜 고정 가격 재현, 가용성 표기에 조회 시각 명시, 조인투어 취소정책 부재 시 "결제 전 확인" 문구, 모바일 뷰에서 배지 접힘 없음, `_meta.warnings` 빈 상태 확인(이번 세션 없음).
