## Baseline 확정값 (세션 4회 이상 완료 시 기입)

| 지표 | 과제 A 중앙값 | 과제 B 중앙값 |
|---|---|---|
| 순 소요시간 (분) | **8분 20초** (n=3: 6:51·8:20·19:21) | 미확정 (시간 데이터 무효 — 아래 주) |
| 엣지케이스 수 | **6** (n=3: 6·6·6) | 해당 없음(과제 B 지표 아님) |
| 상품 근거 수 | **5** (n=3: 4·5·5) | 0 (diagnose는 인용 비필수) |
| 분류 정확도(과제 B 주지표) | — | **3축 12.5/24 (n=2 중앙값)** — B-02 12/24·B-06 13/24 (B-04 클러스터 제외). 축별 중앙값 여정 19.5·마찰 20.5·심각도 18.5 |

- **측정 방침 결정 (2026-07-08, A안 채택)**: 시간 지표(순 소요)는 **과제 A로만** 파일럿 판정에 사용한다. 과제 B는 **품질(분류 정확도)만** 사용하고 시간 비교에서 제외한다.
- 확정일: **과제 A 잠정 확정 2026-07-08** (B-01·B-03·B-05, 순수 공개웹·완전측정 3세션 기준). **과제 B는 시간 미확정·품질만 부분 확보** (2026-07-08 6세션 수집 완료).
- 파일럿 판정 기준: 동형 과제 순 소요 중앙값 ≤ baseline의 50%, **그리고** 품질 부지표 ≥ baseline.
  - → 과제 A 파일럿 목표: **순 소요 ≤ 약 4분 10초**(8:20의 50%) **그리고** 엣지케이스 ≥ 6 · 상품 근거 ≥ 5.
  - → 과제 B는 시간 목표 설정 불가(baseline 시간 무효). 품질 목표만: **분류 정확도 ≥ 3축 12.5/24** 유지.

> **과제 B 시간 미확정 사유**: 3축 전량분류 두 세션 B-02(48초)·B-06(56초) 모두 24건 분류를 5~22초에 완료 → 인간 작업시간 대표성이 없어 시간 median에서 제외. B-04는 클러스터 방식·무시각. 결과적으로 과제 B는 **시간 지표를 하나도 확보하지 못했고 품질(분류 정확도) 지표만** 확보됨. **결정(A안)**: 시간 비교는 과제 A로 한정한다. 과제 B는 품질(분류 정확도)만 사용. 추후 과제 B 시간까지 재확보하려면 참가자에게 "24건을 실제로 읽으며 분류하고 완료 후 시각 기록"을 명확히 안내해 T2 구간이 실제 수 분 경과하도록 재수집.
